# Week 4 - Loops

A C# console application demonstrating arrays, `for` loops, and conditional statements.

## What It Does

This program initializes an array of SpongeBob characters, iterates over them using a traditional `for` loop, and uses `if/else if` statements to display a unique fun fact about each character based on their name.

## How to Run

```bash
cd week-4-loops
dotnet run
```
