# Week 1 - Guided Project (Student Grade Calculator)

A C# console application that calculates and displays student grade averages.

## What It Does

This program stores grades for four students (Sophia, Nicolas, Zahirah, and Jeong) across five assignments, calculates their average scores using decimal division, and displays the results in a formatted table with letter grades.

## How to Run

```bash
cd week-1-guided-project
dotnet run
```
