# Code You Software Development (with C#) Assignments

Minimal C# console apps for CodeYou. Requires the .NET 9 SDK.

## Run

From the repo root:

```bash
dotnet run --project <path>
```

Example:

```bash
dotnet run --project week-1-console-writeline
```
